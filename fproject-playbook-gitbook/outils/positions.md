# 📊 Positions

La page **Positions** affiche tous les tokens que vous détenez actuellement suite aux achats effectués par le bot.

Depuis le menu principal, appuyez sur **📊 Positions**.

### Quand il y a des positions

Chaque position affiche :
* Le nom du token et son contract address (tronqué)
* Le prix d'entrée et le prix actuel
* Le P&L (Profit & Loss) en pourcentage et en SOL
* Des boutons de vente rapide

### Vendre depuis les Positions

Vous pouvez vendre manuellement depuis cette page :
* **25% / 50% / 75% / 100%** — Vente rapide d'un pourcentage de votre position
* **Sell All** — Vend toutes vos positions ouvertes d'un coup

> ⚠️ **Sell All** vend **tout**. Utilisez-le avec précaution.

### Quand la page est vide

Si vous n'avez aucun token en votre possession, la page affiche "Aucune position ouverte". Les tokens achetés par le bot apparaîtront ici automatiquement.

Appuyez sur **🔄 Refresh** pour actualiser la liste.

> 💡 **Conseil** : Vous pouvez aussi vendre directement depuis les **notifications d'achat** — c'est souvent plus rapide que de passer par la page Positions.
