# 👜 Créer & Gérer ses Wallets

Depuis le menu principal, appuyez sur **💰 Wallets** pour accéder à la gestion de vos wallets.

### Vos wallets

Chaque wallet est affiché sous forme de carte avec :
* Le nom du wallet
* Le solde en SOL
* Le nombre de ruggers monitorés sur ce wallet

Appuyez sur un wallet pour accéder à ses options : renommer, voir l'adresse, exporter la clé, activer/désactiver le monitoring, supprimer.

### Créer un wallet

**📥 Import Wallet** — Collez la clé privée d'un wallet Solana existant. Utile si vous avez déjà un wallet que vous utilisez ailleurs.

**✨ Generate Wallet** — Le bot génère un nouveau wallet Solana pour vous. L'adresse et la clé privée vous sont communiquées. **Notez la clé privée immédiatement** — elle ne sera plus affichée.

### Disperse SOL

**💸 Disperse SOL** — Permet de répartir du SOL depuis un wallet vers vos autres wallets. Utile si vous utilisez plusieurs wallets et souhaitez les alimenter rapidement.

### Quel wallet pour quoi ?

Chaque wallet peut avoir ses propres ruggers. Par exemple :
* **Wallet A** — Dédié au sniping de créations de tokens (ruggers = wallets de développeurs)
* **Wallet B** — Dédié au copy trading (ruggers = wallets de traders que vous suivez)

Grâce aux **réglages par rugger**, vous n'avez plus besoin de créer des wallets séparés juste pour avoir des configurations différentes. Un seul wallet avec plusieurs ruggers ayant chacun leurs propres paramètres suffit.
