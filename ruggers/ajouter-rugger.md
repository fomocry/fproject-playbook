# 🎯 Ajouter un Rugger

Un **rugger** est une adresse wallet que le bot surveille pour vous. Ça peut être :
* L'adresse d'un **développeur** qui crée des tokens (sniping)
* L'adresse d'un **trader** dont vous voulez copier les achats (copy trading)

### Comment ajouter un rugger

1. Depuis le menu principal, appuyez sur **🎯 Ruggers**
2. Appuyez sur **➕ Add Rugger**
3. Collez l'adresse du wallet dans le chat
4. Le bot crée le rugger **sans réglage** — il est "nu"
5. Appuyez sur le rugger pour le configurer (tracking, achat, vente, protection)

> 💡 **Conseil** : Si vous avez des Presets sauvegardés, appliquez-en un directement après l'ajout pour gagner du temps.

### 30 Slots par wallet

Chaque wallet peut monitorer jusqu'à **30 adresses** simultanément. Le compteur en haut de la page Ruggers indique le nombre d'adresses utilisées sur le maximum autorisé (ex : 12/30).

Quand vous avez beaucoup de ruggers, la liste est **paginée** pour que le menu reste lisible.

### La liste des ruggers

Sur la page Ruggers, chaque adresse monitorée est affichée avec :
* **Le nom** du rugger (ou l'adresse tronquée si vous ne l'avez pas renommé)
* **Le mode de tracking** actif (New Tokens, Track Buys, etc.)
* **Le statut** ON ou OFF

### Toggles Globaux

Au-dessus des boutons Ajouter/Supprimer, vous trouverez les **toggles globaux** :
* **🟢 Global Auto-Buy ON/OFF** — Active ou désactive l'auto-achat pour TOUS vos ruggers d'un coup
* **🟢 Global Auto-Sell ON/OFF** — Active ou désactive l'auto-vente pour TOUS vos ruggers d'un coup

Pratique pour tout couper le soir et tout réactiver le matin, ou pour arrêter rapidement toute activité du bot.

### Interagir avec un rugger existant

Appuyez sur un rugger dans la liste pour ouvrir sa **page de configuration**. C'est ici que vous pouvez modifier tous ses paramètres.

### Supprimer un rugger

Depuis la page Ruggers, appuyez sur **🗑 Remove** puis sélectionnez l'adresse à retirer du monitoring.

### Renommer un rugger

Depuis la config d'un rugger, appuyez sur **✏️ Rename** pour lui donner un nom personnalisé. C'est fortement recommandé quand vous avez plusieurs adresses — "RugKing_42" est plus clair que "7xKp...3mNf".
