# 🎛️ Presets P1 / P2 / P3

Les presets sont le cœur de la vitesse de l'extension : trois configurations de buy pré-enregistrées, exécutables en un clic depuis n'importe où — feed X, Discord ou page token.

## Le principe

Chaque preset (P1, P2, P3) enregistre un **montant de buy** et ses réglages d'exécution. Quand tu cliques sur un preset, le buy part immédiatement avec cette config, sur ton wallet actif.

L'idée : couvrir tes trois tailles de position habituelles.

```
P1 :  0.1 SOL   → entrée test
P2 :  0.5 SOL   → position standard
P3 :  1 SOL     → conviction
```

Les montants ci-dessus sont un exemple — configure-les selon ta gestion de risque et ta bankroll. Ce sont des conseils de départ, pas des règles.

## Configuration

Ouvre l'extension → **Réglages** → **Quick Buy Presets**. Pour chaque preset, tu définis le montant en SOL et les paramètres d'exécution associés.

Les presets sont synchronisés avec ton compte F Project : tu les retrouves à l'identique partout où ton compte est connecté.

## Bien utiliser les presets

* **P1 petit, toujours.** Ton P1 est celui que tu cliqueras le plus vite, souvent dans un feed qui scroll. Garde-le sur un montant que tu peux perdre sans réfléchir.
* **Wallet actif = wallet scout.** Comme pour le Quick Buy by CA du bot, dédie un wallet aux entrées rapides avec une balance limitée. Tes clics impulsifs ne toucheront jamais tes wallets principaux.
* **Vérifie le MC avant de cliquer.** Un coup d'œil sur Axiom ou GMGN avant de tirer — une seconde de vérification t'évite de rentrer au sommet d'une bougie.

> ⚠️ Pas de menu de confirmation sur les presets injectés dans les feeds : c'est ce qui les rend rapides, c'est aussi ce qui les rend dangereux si tes montants sont mal réglés. Configure d'abord, scrolle ensuite.
