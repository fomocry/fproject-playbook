# 🎁 Referral

Le programme de parrainage F Project vous permet de gagner une commission sur l'activité de vos filleuls.

### Comment ça marche

1. Ouvrez le bot et tapez `/referral`
2. Le bot crée automatiquement un **wallet referral** dédié à vos gains de parrainage
3. Récupérez votre lien de parrainage personnalisé
4. Partagez-le avec vos amis ou votre communauté
5. Quand quelqu'un souscrit à F Project via votre lien, il devient votre filleul

### Combien vous gagnez

Pour chaque filleul qui passe par votre lien, vous recevez :
* **35% du montant de chaque vente** effectuée par ce filleul
* **35% des frais de gas** générés par ce filleul

Ces commissions sont versées sur votre **wallet referral** (séparé de votre wallet de trading).

### Bonus : passer à 50%

Si vous parrainez **plus de 20 personnes**, vos commissions passent automatiquement à :
* **50% du montant des ventes** de vos filleuls
* **50% des frais de gas** de vos filleuls

### Conseils

* Partagez votre lien sur vos réseaux sociaux, groupes Telegram, Discord
* Plus vos filleuls tradent, plus vous gagnez
* Le parrainage est illimité — pas de plafond de filleuls
